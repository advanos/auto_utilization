int get_percentage_value ( double *  p_percentage  ) 

Get cpu utilization percentage from stdin. 

    This function get cpu utilization percentage from stdin, the value 
    transfers by point(double *). When your inputs have any mistake, you
    can continue, except inputting 'EOF'.
    Example:
                double percentage = 0;
                get_percentage_value(&percentage);
参数:
[out] p_percentage The percentage got from stdin will transmit by this point.
返回:
Zero for success, none-zero for failures.
返回值:
0 Successful.
-1 Input paremeters error.
-2 Exit from 'EOF'.
参见:
set_cpu_percentage
注意:
**DO NOT** input NULL for p_percentage.

int multi_threads_run ( cpu_utilization *  cpu_args,
int  size 
)

Spend specified utilization percentages to several cpus. 

    This function calls pthread_create for a cpuset and every thread calls a
    thread_bind_cpu function. Then all the cpus in the cpuset would get some 
    specified cpu utilization more with them.
    Example:
                int size; 
                cpu_utilization * cpu_args;
                multi_threads_run(cpu_args, size);
参数:
[in] cpu_args The cpu number and it's utilization percentage.
[in] size The number of the cpuset's members.
返回:
Zero for success, none-zero for failures.
返回值:
0 Successful.
-1 Input paremeters error.
-2 Thread create faild.
参见:
thread_bind_cpu
注意:
**DO NOT** input NULL for cpu_args or none-positive values for size.

void* thread_bind_cpu ( void *  arg  ) 

Bind a thread with a cpu and this thread will spend some utilization more of the very one. 

    This function should be called by pthread_create. First define 
    a cpu_utilization point, and convert it to void*.
    Example:
                pthread_t * p_tid; 
                cpu_utilization * cpu_args;
                pthread_create(p_tid, NULL, thread_bind_cpu, (void*)(cpu_args));

void *
